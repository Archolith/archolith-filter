Attribution sources
-------------------

Several redaction patterns in `archolith_filter/redact.py` were derived from
publicly documented specifications:

- AWS access key ID format (AKIA/ASIA/AGPA/AIDA/AROA prefix + 16 chars):
  AWS documentation on IAM identifiers.

- JWT format (three base64url segments starting with `eyJ`):
  RFC 7519 (JSON Web Token) section 2 and 3.

- OpenAI API key formats (sk-, sk-proj-, sk-svcacct-, sk-svcacct-underscore,
  sk_proj_underscore):
  OpenAI platform documentation on API keys and project keys.

- Anthropic API key formats (sk-ant-, sk-ant-api03-):
  Anthropic platform documentation on API keys.

- GitHub, GitLab, Slack, Twilio, Stripe, SendGrid, npm, PyPI, Docker Hub,
  Sentry tokens:
  Public vendor documentation for each respective platform.

No code was copied verbatim from those documents — the patterns were
re-derived from the publicly described formats. See the inline comments in
`redact.py` for per-pattern source notes.

- Protocol / TLS / connection-string parsing references in
  `archolith_filter/redact.py` (pattern #10, user:pass credential extraction
  from `scheme://user:pass@host`):
  RFC 3986 (Uniform Resource Identifier) section 3.2.1 (User Information).
